# Z90 Method — אתר נחיתה

דף נחיתה לתוכנית אימון אישי **Z90 Method** — תהליך של 90 יום לגברים ונשים בגילאי 40–60 עם כאבי גב והרצון להרגיש שוב חזקים ובשליטה על הגוף.

## תוכן

- `index.html` — האתר הראשי (עמוד יחיד, עברית RTL): hero עם טופס ליד, שיטת Z90 אינטראקטיבית, המלצות, מסלולי מנוי, שאלות נפוצות, טופס יצירת קשר ומפה.
- `landing.html` — גרסת נחיתה חלופית.
- `zohar-coach.png` — תמונת המאמן, זהר הלוי.

## טכנולוגיה

HTML + CSS + JavaScript סטטי, ללא תלות בבנייה. גופן Heebo ואנימציות AOS נטענים מ-CDN.

## הרצה מקומית

פשוט לפתוח את `index.html` בדפדפן, או להריץ שרת מקומי:

```bash
python3 -m http.server 8000
# ואז לפתוח http://localhost:8000
```
